Project description: LearnTrack is a console-based Student & Course Management. This is standalone java project and can run on console only.

---------------------------------------------------------
How to compile and run: Download all files and folder structure which is java project. 
	two option for compiling 
		1. Create project structure in any Editor and it will get compiled. 
		2. From command prompt run command by navigating to root folder. 
			command: javac -d *.java
