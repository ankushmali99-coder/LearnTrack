What is JDK, JRE, JVM
---------------------------
JDK: Java development kit which will include JVM JRE and also other libraries. JDK to be used for coding and programming 
JRE: JRE is java runtime environment, it will include all libraries which will required to run java class/application.
JVM: JVMstands for java virtual machine. It executes .class files which are converted in bytecode.  

What is bytecode
---------------------------
Bytecode is compiled .class file. It is matchine readable/executable code which also univesal and can be run on any JVM. 


What does “write once, run anywhere” mean (1–2 short paragraphs)
---------------------------
Java is comes with JVM which is able to execute bytecode and that bytecode is universal code can be eassily understood by any matchine and can be executed with the hep of JVM. To run any bytecode there will be no any dependancy on specific operating system. It can run on any operating system.

e.g. 
Java code which is implemented on windows operating system can be executed on any operating system. And this feature of preparing and running bytecode on any machine and without dependancy can be executed on any machine call write once, run anywhere.
